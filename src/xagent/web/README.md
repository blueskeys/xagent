# xagent Web 前端系统

## 系统介绍

xagent Web 前端系统是基于 temp/index.html 倒数第二页设计的现代化 Agent 操作系统界面，提供完整的用户交互体验。

## 功能特性

### 1. 用户认证系统
- **登录页面**: 现代化的渐变背景设计
- **演示账户**: admin/admin 或 demo/demo
- **会话管理**: 基于 localStorage 的简单会话管理

### 2. 主界面 (Dashboard)
- **左侧面板**: 历史记录列表，支持搜索和筛选
- **右侧面板**: 自然语言输入框和快速操作
- **响应式设计**: 支持桌面端和移动端
- **快速模板**: 预设的常用任务模板

### 3. Agent 执行界面
完全参考 temp/index.html 倒数第二页的三面板设计：

#### 左面板 - 自然语言交互
- 显示用户请求和 AI 响应
- 支持补充说明输入
- 实时聊天记录

#### 中间面板 - 可视化执行过程
- **5步骤流程图**: 理解需求 → 数据连接 → 数据分析 → 报告生成 → 交付结果
- **状态可视化**: completed(绿色) / active(蓝色脉搏) / pending(灰色)
- **Human-in-the-loop**: 每个步骤前后都有人工干预按钮(+)
- **SVG 连接线**: 展示步骤间的逻辑关系

#### 右面板 - 实时执行状态
- **终端样式日志**: 黑色背景的命令执行记录
- **进度条**: 实时显示任务执行进度
- **浏览器模拟**: 显示内部系统访问
- **系统指标**: CPU、内存、执行时间的实时监控

### 4. 设计系统
- **颜色主题**: 深色主题，符合现代开发工具风格
- **字体**: Inter 字体家族，优秀的可读性
- **交互动画**: 悬停效果、过渡动画、脉搏动画
- **组件库**: 可复用的卡片、按钮、输入框组件

## 技术栈

- **后端框架**: FastAPI
- **模板引擎**: Jinja2
- **静态文件**: CSS + JavaScript
- **数据存储**: localStorage (演示用)
- **样式系统**: 原生 CSS + CSS 变量
- **响应式设计**: CSS Grid + Flexbox

## 快速开始

1. 确保已安装依赖：
```bash
pip install fastapi uvicorn jinja2
```

2. 启动 Web 服务：
```bash
python start_web.py
```

3. 访问系统：
- 打开浏览器访问: http://localhost:8000
- 使用演示账户登录: admin/admin 或 demo/demo

## 页面流程

1. **登录页面** (`/`) → 输入演示账户
2. **主界面** (`/dashboard`) → 查看历史记录，输入新任务
3. **Agent执行界面** (`/agent?id=xxx`) → 观看三面板执行过程

## 核心特色

### 🎯 完美复刻设计
严格按照 temp/index.html 倒数第二页的视觉设计，包括：
- 浏览器窗口样式的容器设计
- 三面板等宽布局
- 流程步骤的精确位置和连接线
- 终端样式的执行日志
- 系统指标的实时更新

### 🔄 Human-in-the-loop
每个执行步骤都支持人工干预：
- 步骤前干预: 调整输入参数
- 步骤后干预: 审查结果，决定下一步

### 📊 实时状态监控
- 命令执行日志
- 进度条和状态指示
- 系统资源使用情况
- 执行时间计时器

### 📱 响应式设计
- 桌面端: 三面板并排显示
- 移动端: 面板可折叠，优化触控操作
